# Core Layer

Contains shared components that do not belong to a specific feature or usecase.

Folders:
- `constants/` - Color schemes, string constants, assets references.
- `network/` - Base client (Dio) wrapper, interceptors for JWT injection, and standard responses error parser.
- `theme/` - System themes (Light, Dark, high contrast styles).
- `utils/` - Global helper modules.
- `errors/` - Standard failures and exceptions declarations.
