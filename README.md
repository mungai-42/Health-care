# Afya Flow Healthcare Platform

Afya Flow is a production-ready commercial healthcare platform designed for high scalability, robustness, and maintainability.

## Technology Stack
- **Backend**: Node.js, Express, TypeScript, Prisma ORM, PostgreSQL
- **Web Dashboard**: React, TypeScript, Vanilla CSS
- **Mobile Application**: Flutter, Dart

## Workspace Layout
- `backend/` - Node.js microservice API built using Clean Architecture principles, featuring custom logging, comprehensive error handling, and API versioning.
- `web/` - React dashboard app with customizable styles, modular feature architecture, routing configuration, and reusable UI components.
- `mobile/` - Flutter application implementation utilizing clean architecture pattern (separated into Data, Domain, and Presentation layers).

## Setup & Running Guide
Refer to individual service READMEs in `backend/`, `web/`, and `mobile/` for specific run/install guides.
